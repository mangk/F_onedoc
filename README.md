# moredoc-app

![](static/images/logo.png)

`魔豆文库专业版`配套小程序，使用uni-app进行开发。魔豆文库社区版的用户，请自行根据当前提供的小程序源码进行修改。

## 二次开发

1. 下载最新的hbuilderx
2. 将 `config.example.js` 重命名为 `config.js`，并修改 `addr` 配置项
3. 执行 `npm install`，安装相关依赖
4. 修改`manifest.json`的微信小程序appid为您的微信小程序appid
5. 使用`hbuilderx`开发
